# 📊 Stock Risk & Return Analysis

## Overview

This project analyses the performance and risk characteristics of high-growth technology stocks (Tesla, Nvidia, Palantir) using Python. The aim is to evaluate the trade-off between return and risk using statistical and financial metrics.

## Methods

The analysis includes:

* Daily returns and log returns
* Volatility and rolling statistics
* Sharpe and Sortino ratios
* Correlation analysis
* Maximum drawdown
* Value at Risk (VaR)
* Distribution of returns

## Key Insights

* Nvidia demonstrates the strongest risk-adjusted performance
* Tesla shows high volatility without consistent long-term growth
* Palantir exhibits extreme risk with large drawdowns and tail events
* All assets show moderate correlation, indicating shared market exposure

## Tools Used

* Python (NumPy, Pandas, Matplotlib)
* Jupyter Notebook

## Repository Structure

* `analysis.ipynb` – main analysis notebook
* `data/` – stock price datasets

## Conclusion

The results highlight the fundamental trade-off between risk and return in high-growth equities, with Nvidia offering a more favourable balance compared to Tesla and Palantir.
